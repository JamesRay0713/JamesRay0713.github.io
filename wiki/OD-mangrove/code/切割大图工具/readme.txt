********** 切割大图工具 ***********
使用方法：
- 解压压缩包到任何地方；
- 双击“切割大图工具.bat”，系统将自动检测安装python及相关依赖包，最后自动开启小工具的界面；
	- “原图目录”：就是你把想要切割的所有大图集中放起来的那个目录；
	- “切分后存放目录”
	- “输出图片格式”：切割后的所有子图格式将默认改为'.tif'，你也可在这里自定义'.jpg, .png'等格式。
		注意：如果原图是tif格式，输出jpg格式的压缩性最强，速度最快；
		输出png格式的保真能力较强，速度最慢；
		输出tif格式的保真能力最强，速度适中，但后续导入AI模型可能需要进一步的格式转码工作。
	- “子图尺寸”：默认为640*640
	- “重叠比率”：默认20%，即保证相邻两子图有20%的重叠空间。
- 设定好后，即可点击“开始”，等待窗口最终显示"Mission Success!"，完成任务。
